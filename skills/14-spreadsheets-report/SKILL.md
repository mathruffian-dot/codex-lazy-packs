# Excel 月報整合與公式維護

使用者要整理每月 Excel 實績、跨檔對應台數、整合分頁、維護能率或 ST 公式時，先讀 repo 根目錄：

```text
10-Excel月報整合與公式維護.md
```

## 核心規則

1. 先盤點分頁、公式、引用關係，不要直接改主檔。
2. 複雜 `.xlsx` 主檔優先用 Excel 原生方式開啟、計算、儲存、驗證。
3. 舊 `.xls` 來源檔可能橫向有多組欄位，要掃描全部欄位組。
4. 不要刪底層計算分頁；正式維護版用 `Very Hidden`。
5. 每月可變條件集中到 `月別設定`。
6. `整合總覽` 只做閱讀與月份切換。
7. 交付前一定驗證 Excel 可正常開啟，且來源合計與主檔合計一致。

## 必查坑位

- A:M 或後段欄位漏整理：代表只抓到其中一組欄位。
- Q:X 公式存在但值空白：通常是公式快取或重算問題。
- 開檔跳修復：先停止交付，改用 Excel 修復另存並重新驗證。
- 刪分頁造成月份切換失真：改用 Very Hidden，不要刪。
- 使用 XLOOKUP 被轉 `_xludf.XLOOKUP`：改用相容公式或 Excel 原生寫入後驗證。

## 回報格式

回報要包含：

- 來源檔與主檔
- 更新月份與目標欄位
- 來源整理筆數與台數合計
- 主檔寫入筆數與台數合計
- 新增機種 code
- ST 未對應 code
- 可見分頁清單
- 是否通過 Excel 開檔驗證
